# 40+ Simple C Programming Projects 🚀

Kumpulan program sederhana dalam bahasa C yang dirancang untuk mengasah logika pemrograman dan memberikan solusi praktis untuk masalah sehari-hari. Repositori ini mencakup berbagai topik mulai dari dasar input-output, algoritma matematika, hingga manajemen data sederhana.

## 📌 Kategori Program

Untuk memudahkan navigasi, berikut adalah daftar program yang tersedia berdasarkan fungsinya:

### 💰 Keuangan & Bisnis (Finance & Business)
* `diskon.c` & `diskonmember.c` - Kalkulator diskon umum dan membership.
* `pajakpph.c` - Penghitung pajak penghasilan sederhana.
* `cicilan.c` - Estimasi cicilan bulanan.
* `kasir.c` - Sistem kasir sederhana untuk toko.
* `kembalian.c` - Menghitung uang kembalian belanja.
* `tabungan.c` & `tabunganrencana.c` - Simulasi target menabung.
* `splitbill.c` - Pembagi tagihan makan bersama.

### 🚗 Transportasi & Utilitas (Daily Utility)
* `estimasibensin.c` - Hitung konsumsi bahan bakar perjalanan.
* `parkir.c` - Kalkulator tarif parkir berdasarkan durasi.
* `servismobil.c` - Pengingat servis rutin kendaraan.
* `speedconv.c` - Konversi MPH ke KM/Jam.
* `timbangan.c` - Konversi satuan berat (KG, Gram, Ons).
* `alarmtidur.c` - Rekomendasi waktu bangun tidur yang cukup.
* `timer_sederhana.c` - Countdown timer interaktif.

### 📊 Manajemen Data & Pendidikan
* `perpustakaan.c` - Database buku menggunakan Struct.
* `kontakteman.c` - Manajemen daftar kontak sederhana.
* `stokbarang.c` - Sistem cek inventaris gudang.
* `ipsemester.c` - Kalkulator rata-rata nilai akademik.
* `antrean.c` - Simulasi nomor antrean loket.

### 🧮 Logika & Matematika
* `cekprima.c` - Menentukan bilangan prima.
* `faktorial.c` - Perhitungan faktorial secara rekursif.
* `desimalbiner.c` - Konversi sistem bilangan.
* `kuadran.c` - Penentu posisi koordinat titik.
* `swappointer.c` - Penukaran nilai menggunakan Pointer.
* `polasegitiga.c` - Visualisasi pola bintang.
